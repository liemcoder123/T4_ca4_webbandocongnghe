- **Frontend:** Mô tả về các công nghệ frontend được sử dụng (ví dụ: HTML, CSS, JavaScript, React, Angular, Vue, v.v.).
- **Backend:** Mô tả về các công nghệ backend được sử dụng (ví dụ: Node.js, Express, Django, Ruby on Rails, v.v.).
